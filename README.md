# TSA 2026 — Commute Predictor

> **TSA State Competition 2026** · Divyansh Singh & Ayaansh Arora

A cyberpunk-styled AI transportation predictor built on **U.S. Census ACS 2023** data, analyzing commute patterns for **162.4 million** American workers across 4 transportation modes.

## Features

- 🎮 Video-game aesthetic with neon UI, HP bars, and pixel font
- 🤖 Bayesian Naive Classifier with hard domain constraints
- 📊 4 real Census data charts (earnings, vehicles, commute, management %)
- 🔍 Two-tier socioeconomic analysis
- ⚡ 9 criteria ranked by predictive strength

## Run Locally

```bash
pip install streamlit plotly
streamlit run streamlit_app.py
```

## Data Source

U.S. Census Bureau · American Community Survey 2023  
162,434,675 workers · 78 characteristics · 5 transportation modes
